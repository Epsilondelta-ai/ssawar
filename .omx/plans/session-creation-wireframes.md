## 화면 1. 홈

### 데스크탑

```text
+----------------------------------------------------------------------------------+
| ssawar                                              [Explore] [Popular] [Profile] |
+----------------------------------------------------------------------------------+
|                                                                                  |
|                    AI들끼리 한 판 벌여볼까?                                       |
|                                                                                  |
|                    [ 새 세션 시작 ]                                              |
|                                                                                  |
|                    진행자 AI와 참가 AI를 고른 뒤                                 |
|                    빈 세션에서 바로 대화를 시작한다                              |
|                                                                                  |
|   Popular Sessions                                                               |
|   [ 창업 조언 대결 ] [ 누가 더 독설적인가 ] [ 밈 감별 배틀 ]                     |
|                                                                                  |
|   Recommended Lineups                                                            |
|   [ GPT + Claude ] [ GPT + Gemini + Claude ] [ Chaos 4 ]                        |
|                                                                                  |
+----------------------------------------------------------------------------------+
```

### 핵심 포인트
- 홈에서 질문 입력을 강제하지 않는다.
- 가장 눈에 띄는 CTA는 `새 세션 시작`.
- 인기 세션과 추천 라인업은 탐색용이다.

## 화면 4. 제목 자동 생성 이후

### 데스크탑

```text
+----------------------------------------------------------------------------------+
| ssawar | 창업 조언 배틀                      [rename]      [share] [session menu] |
+----------------------------------------------------------------------------------+
| Left Rail                | Main Chat                                              |
|--------------------------+---------------------------------------------------------|
| Participants             | Claude Sonnet (orchestrator)                           |
| Orchestrator             | "오늘 주제는 창업 조언입니다. 먼저 GPT부터..."         |
| - Claude Sonnet          |                                                         |
|                          | GPT-4o                                                 |
| Players                  | "먼저 고객 인터뷰를..."                                |
| - GPT-4o                 |                                                         |
| - Claude                 | Claude                                                 |
| - Gemini                 | "아니다, 유통 구조부터..."                              |
|                          |                                                         |
| Heat                     | Gemini                                                 |
| [=====---]               | "둘 다 틀렸고, 시장 정의가 먼저다."                    |
|                          |                                                         |
| Session Controls         | [ 메세지를 입력하세요...                            ][↑] |
| [ End ] [ Summarize ]    |                                                         |
+----------------------------------------------------------------------------------+
```

### 제목 자동 생성 규칙 노출
- 첫 1~3개의 유의미한 메시지 이후 제목 생성
- 제목 변경 시 작은 토스트:
  - `제목이 '창업 조언 배틀'로 정해졌습니다`
- 유저는 상단 제목 클릭으로 즉시 수정 가능

## 상호작용 흐름

### Flow A. 초간단 시작
1. 홈에서 `새 세션 시작`
2. 생성 패널에서 기본값 확인
3. `Start`
4. `Untitled` 빈 세션 진입
5. 첫 메시지 전송
6. 제목 자동 생성

### Flow B. 오케스트레이터만 변경
1. `새 세션 시작`
2. Orchestrator 드롭다운 변경
3. `Start`
4. 빈 세션 진입

### Flow C. 제목 수동 변경
1. 자동 생성된 제목 노출
2. 유저가 제목 클릭
3. inline edit
4. Enter 또는 blur로 저장

## 빈 상태 카피 초안

- `아직 대화가 시작되지 않았습니다`
- `첫 메시지를 보내면 진행자 AI가 판을 짭니다`
- `제목은 대화가 쌓이면 자동으로 정해집니다`

## 와이어프레임 결정 사항

1. 세션 생성 전에는 질문 입력을 요구하지 않는다.
2. 세션 생성 후 바로 빈 채팅 화면으로 진입한다.
3. 제목은 생성 시 `Untitled`.
4. 제목은 대화 후 자동 생성되며 수동 수정 가능.
5. 홈의 1순위 CTA는 `새 세션 시작`.
6. 각 화면 상태는 스냅샷만 봐도 판별 가능하도록 라벨과 구조를 명확히 둔다.

## 시각 QA 요구사항

### 스냅샷 친화 원칙
- 주요 상태마다 상단 제목과 상태 텍스트가 항상 노출돼야 한다.
- 버튼 라벨은 추상적이지 않고 상태를 직접 설명해야 한다.
- 빈 상태와 진행 상태는 레이아웃만이 아니라 카피로도 구분돼야 한다.
- 자동 제목 생성 전후 상태는 텍스트만 봐도 차이를 알 수 있어야 한다.

### 필수 스냅샷 상태
- 홈
- 세션 생성 패널 오픈
- `Untitled` 빈 세션
- 첫 메시지 직후
- 제목 자동 생성 이후
- 종료 후 요약
